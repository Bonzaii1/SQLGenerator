## Getting Started

To properly set up this project you will have to make sure you have NodeJs Installed. 


### Installation

1. Get an API Key at [https://openai.com/blog/openai-api](https://openai.com/blog/openai-api)
2. Clone the repo
   ```sh
   git clone https://github.com/Bonzaii1/SQLGenerator.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your API in a new `.env`
   ```sh
   OPENAI_API_KEY = 'ENTER YOUR API';
   ```

## Usage

While the application is simple the premise is impactful. This means anyone can now interact with databases without having to learn SQL. It's a game-changer for teams—bridging the gap between technical and non-technical team members, fostering quicker data access, and promoting better collaboration. It's all about making databases more accessible and enabling faster decision-making.
